# MoneyTrace 💸

MoneyTrace is an intelligent financial tracking application designed to simplify personal finance management. Built using an AI-assisted development workflow, this project leverages **Gemini AI** to categorize transactions, generate financial insights, and provide a seamless cross-platform user experience.

## 🚀 The AI-Assisted Workflow
This application was architected and built using **Google Antigravity**—an advanced AI coding agent environment. The development process focused on:
- Architecting the system flow and cross-platform widget trees.
- Writing strict system prompts to enforce Dart's sound null safety and modular UI components.
- Directing the AI agent to integrate the Gemini AI API for core data processing.
- Managing platform-specific code (HTML, Swift, C++) generated by the framework without breaking the core Dart logic.

## 🧠 Core Features
- **Smart Categorization:** Integrates with Gemini AI to automatically parse and categorize raw transaction inputs.
- **Financial Insights:** Uses LLM capabilities to detect spending patterns and offer actionable budgeting advice.
- **Cross-Platform:** Built to compile seamlessly across different environments using a single Dart codebase.

## 🛠 Tech Stack
- **AI Coding Agent:** Google Antigravity
- **AI Integration:** Gemini AI API
- **Core Language:** Dart
- **Framework Ecosystem:** Flutter (with HTML, C++, Swift, and CMake for native targeting)
